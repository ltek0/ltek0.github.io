Why AWS

I prefer using AWS with a full virtual machine to host my website, even though there are serverless options like Hostinger available. I have experience with Linux web hosting and have a testing setup at home, which makes it easier for me to test and deploy changes to the cloud. I'll prabably switch to something like lambda or something server less as the price goes up.
